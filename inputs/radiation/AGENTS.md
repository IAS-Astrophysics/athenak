# Directory Guide

## Role
Radiation transport and radiation-coupled fluid example inputs.

## Important Files
- `beam.athinput`, `bh_beam.athinput`, `shadow.athinput`, `snake.athinput`: directional transport and source-term style cases.
- `hohlraum_2d.athinput`, `rad_diffusion.athinput`, `relax.athinput`: diffusion and relaxation problems.
- `rad_fm_sane_8_4.athinput`: GR accretion-style radiation run.
