# Directory Guide

## Role
Inline Riemann solvers for the dynamical-spacetime GRMHD module.

## Important Files
- `llf_dyn_grmhd.hpp`, `hlle_dyn_grmhd.hpp`: supported approximate solvers.
- `flux_dyn_grmhd.hpp`: shared flux assembly utilities used by the solver variants.
