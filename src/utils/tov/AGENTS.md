# Directory Guide

## Role
Tolman-Oppenheimer-Volkoff helper routines for neutron-star initial data and related utilities.

## Important Files
- `tov.hpp`, `tov.cpp`: main TOV solution utilities.
- `tov_polytrope.hpp`, `tov_piecewise_poly.hpp`, `tov_tabulated.hpp`: EOS-specific TOV helper variants.
- `tov_utils.hpp`: shared support routines.
